## 자세한 설명 \*

**Metronome & Chords**는 음악가와 연주자를 위한 전문적인 메트로놈 및 코드 연습 앱입니다. 정확한 템포 연습과 코드 학습을 한 번에 제공하는 올인원 음악 연습 도구입니다.

### 🎵 주요 기능

#### 정밀한 메트로놈

- **넓은 BPM 범위**: 20~300 BPM까지 정밀하게 조절 가능
- **직관적인 조작**: 슬라이더와 +/- 버튼으로 쉬운 BPM 조절
- **타임 시그니처 설정**: Beat/bar와 Click/beat를 각각 1~16까지 자유롭게 설정
- **템포 마킹**: Larghissimo부터 Prestissimo까지 15가지 클래식 템포 마킹 제공
- **시각적 비트 표시**: 현재 비트를 실시간으로 시각적으로 확인
- **프리셋 저장**: 자주 사용하는 설정을 저장하고 빠르게 불러오기
- **햅틱 피드백**: iOS에서 비트마다 진동 피드백 제공

#### 코드 연습 도구

- **다양한 코드 타입**: M7, m7, 7, m7(b5), dim7, mM7, aug7, sus4 등 8가지 코드 타입 지원
- **12음계 지원**: 모든 반음계(크로매틱) 지원
- **텐션 추가**: 9, 11, 13 등 다양한 텐션으로 코드 확장 가능
- **커스터마이징**: 원하는 노트, 코드, 텐션만 선택하여 연습
- **프리셋 시스템**: 연습 설정을 저장하고 관리

#### 사용자 친화적 인터페이스

- **다크 모드**: 눈의 피로를 줄이는 어두운 테마
- **직관적인 디자인**: 깔끔하고 사용하기 쉬운 UI/UX
- **빠른 접근**: 주요 기능에 즉시 접근 가능

### 🎯 누구를 위한 앱인가요?

- **악기 연주자**: 기타, 피아노, 드럼 등 모든 악기 연주자
- **음악 학생**: 리듬 감각과 코드 이해를 향상시키고 싶은 학생
- **프로 뮤지션**: 정확한 템포 연습이 필요한 프로페셔널 뮤지션
- **작곡가**: 정확한 템포로 작품을 만들고 싶은 작곡가
- **연습생**: 일정한 템포로 연습하고 싶은 모든 음악 애호가

### ✨ 왜 Metronome & Chords를 선택해야 할까요?

1. **정확성**: 정밀한 BPM 제어로 완벽한 템포 연습
2. **편의성**: 두 가지 핵심 기능을 하나의 앱에서 제공
3. **유연성**: 다양한 설정 옵션으로 모든 연습 스타일에 대응
4. **효율성**: 프리셋 기능으로 빠른 설정 전환
5. **전문성**: 프로페셔널한 기능과 사용자 경험

### 📱 사용 방법

**메트로놈 사용하기:**

1. 원하는 BPM을 슬라이더나 버튼으로 설정
2. Beat/bar와 Click/beat를 선택
3. 템포 마킹을 선택하거나 직접 BPM 입력
4. 재생 버튼을 눌러 시작
5. 자주 사용하는 설정은 프리셋으로 저장

**코드 연습하기:**

1. 연습하고 싶은 노트 선택
2. 연습할 코드 타입 선택
3. 필요시 텐션 추가
4. 연습 시작

### 🔧 기술 사양

- 코드 연습 시 한 번에 너무 많은 옵션을 선택하지 말고 단계적으로 연습
- 프리셋을 활용하여 연습 세션별로 다른 설정 관리
- 정기적인 연습으로 리듬 감각과 코드 이해도 향상
