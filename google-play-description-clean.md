## 영어(미국) - 설명

Metronome & Chords is a professional metronome and chord practice app designed for musicians and performers. An all-in-one music practice tool that provides accurate tempo practice and chord learning in one place.

Key Features

Precision Metronome

- Wide BPM Range: Precise control from 20 to 300 BPM
- Intuitive Controls: Easy BPM adjustment with slider and +/- buttons
- Time Signature Settings: Freely set Beat/bar and Click/beat from 1 to 16 each
- Tempo Markings: 15 classic tempo markings from Larghissimo to Prestissimo
- Visual Beat Display: Real-time visual indication of current beat
- Preset Saving: Save frequently used settings and load them quickly
- Haptic Feedback: Vibration feedback for each beat on iOS

Chord Practice Tools

- Various Chord Types: Support for 8 chord types including M7, m7, 7, m7(b5), dim7, mM7, aug7, sus4
- 12-Tone Support: Support for all semitones (chromatic)
- Tension Addition: Expand chords with various tensions like 9, 11, 13
- Customization: Practice by selecting only desired notes, chords, and tensions
- Preset System: Save and manage practice settings

User-Friendly Interface

- Dark Mode: Eye-friendly dark theme
- Intuitive Design: Clean and easy-to-use UI/UX
- Quick Access: Instant access to key features

Who Is This App For?

- Instrumentalists: All instrumentalists including guitar, piano, drums
- Music Students: Students who want to improve rhythm sense and chord understanding
- Professional Musicians: Professional musicians who need accurate tempo practice
- Composers: Composers who want to create works with precise tempo
- Practitioners: All music enthusiasts who want to practice at a consistent tempo

Why Choose Metronome & Chords?

1. Accuracy: Perfect tempo practice with precise BPM control
2. Convenience: Two essential features in one app
3. Flexibility: Adapt to all practice styles with various setting options
4. Efficiency: Quick setting changes with preset functionality
5. Professionalism: Professional features and user experience

How to Use

Using the Metronome:

1. Set desired BPM using slider or buttons
2. Select Beat/bar and Click/beat
3. Choose tempo marking or enter BPM directly
4. Press play button to start
5. Save frequently used settings as presets

Practicing Chords:

1. Select notes you want to practice
2. Choose chord types to practice
3. Add tensions if needed
4. Start practicing

Technical Specifications

- Platform: iOS, Android
- Languages: Korean, English
- Interface: Dark mode support
- Storage: Local preset storage (no cloud sync required)

Tips

- When practicing with metronome, start with slow tempo and gradually increase speed
- For chord practice, don't select too many options at once, practice step by step
- Use presets to manage different settings for each practice session
- Improve rhythm sense and chord understanding through regular practice
